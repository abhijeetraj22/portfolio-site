# Abhijeet Raj — Portfolio Website

This repository contains a static portfolio website built for `abhijeetraj22`.  
It’s ready to be hosted on **GitHub Pages**, **Vercel**, **Netlify**, or **Azure Static Web Apps**.

## What’s included
- `index.html` — Home (hero, featured projects, GitHub stats, contact)
- `projects.html` — Full projects listing (all GitHub repos)
- `assets/css/styles.css` — Styling
- `assets/js/particles-config.js` — animated background config
- `assets/js/main.js` — theme toggle, form handlers
- `assets/images/logo.png` — add your logo here
- `assets/images/resume.pdf` — add your resume here

## Quick setup (GitHub Pages)
1. Create a repo named `abhijeetraj22.github.io` or use `abhijeetraj22/portfolio-site` and enable Pages from `gh-pages` branch or `main`/`docs` folder.
2. Push all files to the repo.
3. In GitHub repo → Settings → Pages → select branch and folder → Save.
4. Your site will be available at `https://abhijeetraj22.github.io/` (or `https://<username>.github.io/<repo>/`).

## Contact form (Formspree)
This template uses Formspree for form handling. To enable:
1. Create a free Formspree account.
2. Create a form and copy the form endpoint (looks like `https://formspree.io/f/XXXXX`).
3. Replace the `action` attribute in `index.html` `form` with your endpoint.
